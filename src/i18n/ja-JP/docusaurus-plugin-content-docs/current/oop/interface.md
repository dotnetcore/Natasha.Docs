---
title: "interface"
---

クラスをすばやく作成する：

```cs

クラスを作成し、型
var type = new OopOperator() を
  。 Namespace<string>()
  . Interface()
  . Access(Access.None)
  . Name("TestUt3")
  . Body(@"static void Test(); ")
  . GetType();

```

または

```cs

クラスを作成し、型
var type = new NInterface() を取得
  。 Namespace<string>()
  . Access(Access.None)
  . Name("TestUt3")
  . Body(@"static void Test(); ")
  . GetType();

```
